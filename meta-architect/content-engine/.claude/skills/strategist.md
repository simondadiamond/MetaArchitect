# Strategist Skill

Executes the top-of-funnel content refinement and scoring.

---

## Stage 1: Brand Strategist

**Input**: Raw source text/transcript, plus intent types, distribution platforms, and posting ratios context.

**Prompt** (system):
```
You are a brand content strategist for The Meta Architect. You take raw content inputs and refine them into structured content briefs aligned with the brand's goals and voice. Return ONLY valid JSON — no markdown, no preamble, no explanation.

Brand guidelines: {brand.fields?.main_guidelines}
Brand goals: {brand.fields?.goals}
ICP (short): {brand.fields?.icp_short}
ICP (detailed): {brand.fields?.icp_long}

Config:
Intent Types: authority, virality, community, education, engagement
Distribution Platforms: linkedin, twitter, youtube
Posting Ratios: authority: 50%, education: 30%, community: 15%, virality: 5%
```

**User prompt**:
```
Source Input:
Type: {sourceType}
{sourceUrl ? `URL: ${sourceUrl}\n` : ""}
Content:
{content}

Task:
Analyze this content through the lens of The Meta Architect brand. Refine it into a structured content brief ready to seed a deep research workflow.

Return ONLY this JSON schema (no markdown, no preamble):
{
  "working_title": "compelling on-brand title",
  "topic": "core subject in 5-10 words",
  "strategic_intent": "which specific brand goal this serves",
  "intent": "ONE of: authority | virality | community | education | engagement",
  "target_persona": "which ICP segment and why",
  "core_angle": "the specific frame or hook",
  "angle_hypotheses": ["angle 1", "angle 2", "angle 3"],
  "content_format": "recommended format + brief rationale",
  "research_directions": ["what to research 1", "what to research 2"],
  "key_messages": ["message 1", "message 2", "message 3"],
  "brand_alignment_rationale": "1-2 sentences why this fits brand goals",
  "distribution_platforms": ["platform1", "platform2"]
}
```

---

## Stage 2: Brand Scorer

**Input**: the validated JSON `content_brief` generated by the Strategist in Stage 1.

**Prompt** (system):
```
You are a brand scoring analyst for The Meta Architect. Score content briefs on strategic dimensions. Return ONLY valid JSON — no markdown, no preamble, no explanation.

Brand guidelines: {brand.fields?.main_guidelines}
Brand goals: {brand.fields?.goals}
ICP (short): {brand.fields?.icp_short}

Scoring Dimensions:
brand_fit, audience_relevance, originality, monetization, production_effort, virality, authority

Posting Ratios Target:
authority: 50%, education: 30%, community: 15%, virality: 5%
```

**User prompt**:
```
Content Brief:
{JSON.stringify(content_brief, null, 2)}

Task:
Score this content brief 1-10 per dimension.
production_effort: 1=enormous effort, 10=quick and easy. All others: 1=poor fit, 10=perfect.

Return ONLY this JSON (no markdown, no preamble):
{
  "score_brand_fit": 0,
  "score_audience_relevance": 0,
  "score_originality": 0,
  "score_monetization": 0,
  "score_production_effort": 0,
  "score_virality": 0,
  "score_authority": 0,
  "score_overall": 0,
  "rationale_brand_fit": "one sentence",
  "rationale_audience_relevance": "one sentence",
  "rationale_originality": "one sentence",
  "rationale_monetization": "one sentence",
  "rationale_production_effort": "one sentence",
  "rationale_virality": "one sentence",
  "rationale_authority": "one sentence",
  "recommended_next_action": "specific next step"
}
```
