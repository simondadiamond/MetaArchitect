# Discovery map — call answers → install choices

Fill this during/after the discovery call. It is the entire input session 0 needs.
File the completed copy in the client's record (CC → Clients once it exists; lead notes
until then) and feed the call log to the Convert queue.

## The questions (30 min)

1. Walk me through yesterday, start to finish. (listen for: repeated typing, retyping,
   copy-paste between apps, paper)
2. What do you sell and what does it cost? (tests whether pricing exists in writing —
   if they hesitate, session 0 budgets 20 min for `pricing.md`)
3. Where do customers come from and where do conversations happen? (channels; also the
   follow-up gap: "what happens after a job finishes?" — the answer is usually "nothing")
4. What task do you put off every week? (the first-skill candidate and the demo moment)
5. Who else touches the business? (staff/spouse: affects how-we-work.md and handover)
6. What tools now? (calendar, invoicing, POS: we integrate by copy-paste conventions,
   not APIs, in v1 — set that expectation on the call)
7. Anything you must never see leave the building? (privacy red lines; if health/legal/
   finance data appears anywhere → professional-practice variant, full stop)
8. FR, EN, or both with customers? (sets LOCALE and draft-language rules)

## Mapping

| Signal | Choice |
|---|---|
| Sells hours/jobs/projects | variant: service-provider |
| Sells things (retail, food, maker, e-com) | variant: retail-product |
| Regulated/sensitive client data anywhere | variant: professional-practice (wins all ties) |
| Hated task = email pile | first skills: inbox-triage, follow-up-writer, weekly-review |
| Hated task = quoting/estimates | first skills: quote-builder, inbox-triage, weekly-review |
| Hated task = billing/chasing money | first skills: invoice-draft, follow-up-writer, weekly-review |
| No written pricing | add 20 min to session 0 for pricing.md before ANY skill |
| Hybrid business | pick the dominant revenue side; borrow single rules from the other variant inline |

Always exactly **3 skills in session 0**. More installs worse: each skill must run once
on real work before the session ends, and 90 minutes fits three runs. weekly-review is
in every set unless the owner fights for something else; it's the habit anchor.
